# UNO Demo — GameMaker

![Screenshot](screenshot.webp)

2D desktop/web client for [Turn-Based UNO Demo](../README.md) built with [GameMaker](https://gamemaker.io/) and [Colyseus GameMaker SDK](https://docs.colyseus.io/getting-started/gamemaker).

## Setup

1. Open `UnoGame/UnoGame.yyp` in GameMaker Studio 2 (2024.x or later)
2. Download and import the [Colyseus GameMaker SDK](https://docs.colyseus.io/getting-started/gamemaker)
3. Press **F5** or click Run

Make sure the [game server](../server/) is running on port 2567.

## Controls

- **Click** — Play a card
- **Hover** — Preview playable cards
