# UNO Demo — React + R3F

![Screenshot](screenshot.webp)

3D web client for [Turn-Based UNO Demo](../README.md) built with [React](https://react.dev/), [React Three Fiber](https://docs.pmnd.rs/react-three-fiber), [Colyseus TypeScript SDK](https://docs.colyseus.io/getting-started/typescript), and Vite.

## Setup

```bash
npm install
npm run dev
```

Make sure the [game server](../server/) is running on port 2567.

## Controls

- **Click** — Play a card
- **Hover** — Preview playable cards
