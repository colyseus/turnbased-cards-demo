# UNO Demo — Haxe + Heaps

![Screenshot](screenshot.webp)

2D web/desktop client for [Turn-Based UNO Demo](../README.md) built with [Heaps](https://heaps.io/), [Colyseus Haxe SDK](https://docs.colyseus.io/getting-started/haxe), and Haxe.

## Setup

```bash
haxe build.js.hxml
```

Open `index.html` in a browser. Make sure the [game server](../server/) is running on port 2567.

## Controls

- **Click** — Play a card
- **Hover** — Preview playable cards
